# Node-MVC-Skeleton

Base structure project nodejs

<img src="public/images/express.png" alt="drawing" width="400"/>

Clone the project
```
git clone https://github.com/benoitpetit/node-mvc-skeleton
```
Install dependencies
```
npm install
```
Run the server node
> with nodemon
```
npm run dev
```
> with node
```
npm run start
```
